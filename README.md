# Personal Website (NextJS/Vercel)

## About :round_pushpin:
Clean responsive Personal Website bulit using NextJS and hosted on Vercel.

## Features :bulb:
- Responsive UI
- Virtual Dom
- Static Site Generation (SSG)
- API Routes
- Client-Side Rendering (CSR)
- Image Optimization
- Webpack and SWC Compiler
- Vercel Integration
